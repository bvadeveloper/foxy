
# Foxy (@vulns_search_bot)
Small high scalable Telegram bot for RHT (Red Hat Teams) for gathering and analyzing target sources information in your smartphone.

## Local run

```bash
Set environment variables:

API_KEY=<key>
RMQ_USER=<user>>
RMQ_PASS=<pass>

cd foxy/
make up_locally
```

## Local cleanup

```bash
cd foxy/
make cleanup_all_locally
```
